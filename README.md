# WIFI-Remote-Controller

Controll esp32 via WIFI

Arm servo PIN Out ->

servo 1 - GPIO 18

servo 2 - GPIO 19

Rudder servo - GPIO 21

Sensor 1 (Left):

VCC → 5V/
Trig → GPIO25/
Echo → GPIO26/
GND → GND

Sensor 2 (Bottom):

VCC → 5V/
Trig → GPIO27/
Echo → GPIO14/
GND → GND

Sensor 3 (Right):

VCC → 5V/
Trig → GPIO12/
Echo → GPIO13/
GND → GND
